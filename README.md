# shotweet

shotweet turns Twitter/X post URLs into vertical PNG posters for sharing. Design and API details live in `docs/superpowers/specs/2026-04-18-shotweet-design.md`.

Generated posters include a small footer attribution: **shotweet from xxlemon**, with **xxlemon** linking to the GitHub repository at `https://github.com/0xNMLSS/shotweet` (configurable via `SHOTWEET_BRAND_REPO_URL` once implementation exists).
